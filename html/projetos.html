Descrição do projeto

Este sistema foi desenvolvido usando a combinação de C# .NET 6, Entity Framework Core, Identity, JWT (JSON Web Tokens) e MySQL para o back end, enquanto o front end foi construído com HTML, CSS, JavaScript e Bootstrap.

Nesse sistema é possível realizar:

Cadastro de Clientes: O sistema permite o cadastro e gerenciamento de clientes, facilitando o acompanhamento de informações essenciais e o aprimoramento do atendimento ao cliente.

Controle de Vendas: O gerenciamento de vendas é simplificado com recursos para criar, editar e controlar as vendas realizadas. Os usuários podem efetuar abatimentos ou parcelamentos, garantindo flexibilidade nas transações.

Autenticação e Autorização: Para garantir a segurança e a privacidade dos dados, implementei autenticação de usuário por meio do Identity e JWT, garantindo que apenas usuários autenticados tenham acesso às funcionalidades do sistema.

Histórico de Pagamentos: Os registros de histórico de pagamentos permitem um acompanhamento detalhado de todas as transações, auxiliando na análise financeira e no histórico do cliente.

Registro de Lotes: A funcionalidade de registro de lotes oferece controle adicional, permitindo um gerenciamento eficaz de produtos.

Este projeto foi desenvolvido de forma 100% personalizada de acordo com a necessidade de um cliente, embora existam diversas formas de chegar no mesmo resultado (algumas melhores e outras nem tanto), todas as funcionalidades e comportamentos foram desenvolvidas totalmente de acordo com o que foi combinado com o cliente. Aproveitei a flexibilidade e o desempenho oferecidos pela tecnologia .NET 6 para criar uma solução completa que atende às necessidades de gerenciamento de vendas de uma loja.

Descrição do projeto

Este projeto é baseado em um controle para cadastro de contatos, onde é possível cadastrar novos contatos e usuários para realizarem o login nesse sistema. Todos os contatos e usuários cadastrados são armazenados em uma base de dados rodando em um servidor local. O projeto segue o padrão arquitetural MVC, onde na mesma aplicação foram desenvolvidas todas as views do front-end usando HTML, CSS e JavaScript, Bootstrap, JQuery; como também todo o back-end usando .NET, ASP.NET Core, Entity Framework, SQL Server. Apenas usuários cadastrados na base de dados podem acessar o sistema para consultar os contatos cadastrados. Dentro do sistema é possivel além de cadastrar usuários e contatos, consultar a base de dados, alterar os dados, deletar registros.
