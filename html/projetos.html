Descrição do projeto

O projeto agenda é um projeto de estudo para consolidar os conhecimentos adquiridos em cursos de especialização. O projeto Agenda se trata de uma agenda virtual onde é capaz adicionar, alterar e remover eventos e compromissos em um mural que fica visível na página, todo esse processo roda em um servidor local onde são armazenadas as informações dos eventos/compromissos cadastrados, como por exemplo: o nome do evento, a data e uma descrição sobre o mesmo. O projeto foi desenvolvido com as seguintes tecnologias: C#, ASP .NET Core, Entity framework, MySQL para o back-end; HTML, CSS, JavaScript, Axios framework para o front-end; e Postman para auxiliar nos testes das requisições.
